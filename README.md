# 記帳本（Account Book）

📌 **記帳本** 是一個使用 **Node.js + Express + MongoDB** 打造的記帳系統，讓使用者可以記錄 **收入與支出**，提供 **新增、刪除、查詢記帳紀錄** 的功能。

---

## 🚀 功能特色

✅ **新增記帳紀錄**（記錄金額、類別、日期）  
✅ **刪除記帳紀錄**  
✅ **時間格式化（Moment.js）**  
✅ **MongoDB 存取數據（Mongoose）**  
✅ **使用 EJS 來渲染前端頁面**  

---

## 🛠 使用的技術

- **後端**：`Node.js` + `Express`
- **前端模板引擎**：`EJS`
- **資料庫**：`MongoDB` + `Mongoose`
- **時間處理**：`Moment.js`
- **日誌工具**：`Morgan`（記錄 API 請求）
- **Cookie 解析**：`cookie-parser`
- **靜態資源管理**：`express.static`
